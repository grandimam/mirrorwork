# Appendix C: Memory Layout Diagrams

## PyObject Structure

All Python objects start with a common header:

```
┌─────────────────────────────────────────────────────────────┐
│ PyObject (Base of all objects)                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ┌─────────────────────────────────────────┐                │
│  │ Py_ssize_t ob_refcnt        (8 bytes)   │ Reference count│
│  ├─────────────────────────────────────────┤                │
│  │ PyTypeObject *ob_type       (8 bytes)   │ Type pointer   │
│  └─────────────────────────────────────────┘                │
│                                                             │
│  Total: 16 bytes on 64-bit systems                          │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## PyVarObject Structure

Variable-size objects (list, tuple, bytes, etc.):

```
┌─────────────────────────────────────────────────────────────┐
│ PyVarObject (Variable-size objects)                         │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ┌─────────────────────────────────────────┐                │
│  │ Py_ssize_t ob_refcnt        (8 bytes)   │                │
│  ├─────────────────────────────────────────┤                │
│  │ PyTypeObject *ob_type       (8 bytes)   │                │
│  ├─────────────────────────────────────────┤                │
│  │ Py_ssize_t ob_size          (8 bytes)   │ Number of items│
│  └─────────────────────────────────────────┘                │
│                                                             │
│  Total: 24 bytes on 64-bit systems                          │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## Integer (PyLongObject)

```
┌─────────────────────────────────────────────────────────────┐
│ PyLongObject (Arbitrary precision integer)                  │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ┌─────────────────────────────────────────┐                │
│  │ ob_refcnt                   (8 bytes)   │                │
│  ├─────────────────────────────────────────┤                │
│  │ ob_type                     (8 bytes)   │                │
│  ├─────────────────────────────────────────┤                │
│  │ ob_size                     (8 bytes)   │ Sign + digit   │
│  │                                         │ count          │
│  ├─────────────────────────────────────────┤                │
│  │ ob_digit[0]                 (4 bytes)   │ First digit    │
│  ├─────────────────────────────────────────┤ (30-bit)       │
│  │ ob_digit[1]                 (4 bytes)   │                │
│  ├─────────────────────────────────────────┤                │
│  │ ...                                     │                │
│  └─────────────────────────────────────────┘                │
│                                                             │
│  Small integers (-5 to 256): Pre-allocated, shared          │
│  ob_size: Negative for negative numbers                     │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## String (PyUnicodeObject)

```
┌─────────────────────────────────────────────────────────────┐
│ PyUnicodeObject (Unicode string - compact form)             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ┌─────────────────────────────────────────┐                │
│  │ ob_refcnt                   (8 bytes)   │                │
│  ├─────────────────────────────────────────┤                │
│  │ ob_type                     (8 bytes)   │                │
│  ├─────────────────────────────────────────┤                │
│  │ length                      (8 bytes)   │ Char count     │
│  ├─────────────────────────────────────────┤                │
│  │ hash                        (8 bytes)   │ Cached hash    │
│  ├─────────────────────────────────────────┤                │
│  │ state (kind, compact, etc.) (4 bytes)   │ Encoding flags │

│  Kind encoding:                                             │

## List (PyListObject)

```
┌─────────────────────────────────────────────────────────────┐
│ PyListObject (Dynamic array)                                │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ┌─────────────────────────────────────────┐                │
│  │ ob_refcnt                   (8 bytes)   │                │
│  ├─────────────────────────────────────────┤                │
│  │ ob_type                     (8 bytes)   │                │
│  ├─────────────────────────────────────────┤                │
│  │ ob_size                     (8 bytes)   │ Current length │
│  ├─────────────────────────────────────────┤                │
│  │ **ob_item                   (8 bytes)   │ Pointer to     │
│  │                                         │ array          │
│  ├─────────────────────────────────────────┤                │
│  │ allocated                   (8 bytes)   │ Allocated size │
│  └─────────────────────────────────────────┘                │
│                                                             │
│  ob_item points to:                                         │
│  ┌─────┬─────┬─────┬─────┬─────┬───────────┐                │
│  │ ptr │ ptr │ ptr │ ptr │ ... │ (unused)  │                │
│  └─────┴─────┴─────┴─────┴─────┴───────────┘                │
│  ↓     ↓     ↓                                              │
│  PyObject* pointers to elements                             │
│                                                             │
│  Growth pattern: 0, 4, 8, 16, 24, 32, 40, 52, 64, ...       │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

┌─────────────────────────────────────────────────────────────┐

├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ┌─────────────────────────────────────────┐                │
│  │ ob_refcnt                   (8 bytes)   │                │
│  ├─────────────────────────────────────────┤                │
│  │ ob_type                     (8 bytes)   │                │
│  ├─────────────────────────────────────────┤                │
│  │ ob_size                     (8 bytes)   │ Number of items│
│  ├─────────────────────────────────────────┤                │

│  ├─────────────────────────────────────────┤                │

│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## Dictionary (PyDictObject)

```
┌─────────────────────────────────────────────────────────────┐
│ PyDictObject (Hash table)                                   │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ┌─────────────────────────────────────────┐                │
│  │ ob_refcnt                   (8 bytes)   │                │
│  ├─────────────────────────────────────────┤                │
│  │ ob_type                     (8 bytes)   │                │
│  ├─────────────────────────────────────────┤                │
│  │ ma_used                     (8 bytes)   │ # of items     │
│  ├─────────────────────────────────────────┤                │
│  │ ma_version_tag              (8 bytes)   │ Version (3.6+) │
│  ├─────────────────────────────────────────┤                │
│  │ *ma_keys                    (8 bytes)   │ Keys object    │
│  ├─────────────────────────────────────────┤                │
│  │ **ma_values                 (8 bytes)   │ Values (split) │
│  └─────────────────────────────────────────┘                │
│                                                             │
│  Compact dict layout (3.6+):                                │
│  ┌─────────────────────────────────────────────────────┐    │
│  │ Keys Object (PyDictKeysObject)                      │    │
│  ├─────────────────────────────────────────────────────┤    │
│  │ ┌───────────────────┐                               │    │
│  │ │ Sparse indices    │ [0,1,-1,2,-1,3,...] O(1) lookup│   │
│  │ │ (1/2/4/8 bytes)   │                               │    │
│  │ └───────────────────┘                               │    │
│  │ ┌───────────────────┐                               │    │
│  │ │ Dense entries     │ [(hash,key,value), ...]       │    │
│  │ │                   │ Maintains insertion order     │    │
│  │ └───────────────────┘                               │    │
│  └─────────────────────────────────────────────────────┘    │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## Frame Object (PyFrameObject)

```
┌─────────────────────────────────────────────────────────────┐
│ PyFrameObject (Execution frame) - Python 3.11+              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ┌─────────────────────────────────────────┐                │
│  │ ob_refcnt                               │                │
│  ├─────────────────────────────────────────┤                │
│  │ ob_type                                 │                │
│  ├─────────────────────────────────────────┤                │
│  │ f_back              (PyFrameObject *)   │ Previous frame │
│  ├─────────────────────────────────────────┤                │
│  │ f_code              (PyCodeObject *)    │ Code object    │
│  ├─────────────────────────────────────────┤                │
│  │ f_builtins          (PyDictObject *)    │ Builtins dict  │
│  ├─────────────────────────────────────────┤                │
│  │ f_globals           (PyDictObject *)    │ Global dict    │
│  ├─────────────────────────────────────────┤                │
│  │ f_locals            (PyObject *)        │ Local vars     │
│  ├─────────────────────────────────────────┤                │
│  │ f_trace             (PyObject *)        │ Trace function │
│  ├─────────────────────────────────────────┤                │
│  │ f_lasti             (int)               │ Last instr     │
│  ├─────────────────────────────────────────┤                │
│  │ f_lineno            (int)               │ Current line   │
│  └─────────────────────────────────────────┘                │
│                                                             │
│  Python 3.11+ changes:                                      │
│  - Localsplus array moved to separate allocation            │
│  - Frame objects created lazily                             │
│  - _PyInterpreterFrame used internally                      │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## Memory Pools (pymalloc)

```
┌─────────────────────────────────────────────────────────────┐
│ pymalloc Memory Organization                                │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  Arena (256 KB)                                             │
│  ┌─────────────────────────────────────────────────────┐    │
│  │ Pool │ Pool │ Pool │ Pool │ ... │ Pool │ Pool │ Pool│    │
│  │ 4KB  │ 4KB  │ 4KB  │ 4KB  │     │ 4KB  │ 4KB  │ 4KB │    │
│  └─────────────────────────────────────────────────────┘    │
│    ↓                                                        │
│  Pool (4 KB)                                                │
│  ┌─────────────────────────────────────────────────────┐    │
│  │ Header (48 bytes)                                   │    │
│  │ ┌─────────────────────────────────────────────────┐ │    │
│  │ │ ref.count │ freeblock │ nextpool │ prevpool │..│ │    │
│  │ └─────────────────────────────────────────────────┘ │    │
│  │ Blocks (fixed size per pool)                        │    │
│  │ ┌────┬────┬────┬────┬────┬────┬────┬────┬────┬────┐ │    │
│  │ │ 16 │ 16 │ 16 │ 16 │ 16 │ 16 │ 16 │ 16 │ 16 │ ...│ │    │
│  │ └────┴────┴────┴────┴────┴────┴────┴────┴────┴────┘ │    │
│  └─────────────────────────────────────────────────────┘    │
│                                                             │
│  Size Classes:                                              │
│  ┌──────────────────────────────────────────────────────┐   │
│  │ Index │ Block Size │ Typical Objects                 │   │
│  ├───────┼────────────┼─────────────────────────────────┤   │
│  │   0   │    8       │ (unused on 64-bit)              │   │
│  │   1   │   16       │ Small tuples                    │   │
│  │   2   │   24       │ Small ints, floats              │   │
│  │   3   │   32       │ Small strings                   │   │
│  │  ...  │   ...      │ ...                             │   │
│  │  31   │  512       │ Largest pymalloc object         │   │
│  └──────────────────────────────────────────────────────┘   │
│                                                             │
│  Objects > 512 bytes use system malloc                      │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## Object Instance Layout

```
┌─────────────────────────────────────────────────────────────┐
│ Regular Instance (with __dict__)                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ┌─────────────────────────────────────────┐                │
│  │ ob_refcnt                   (8 bytes)   │                │
│  ├─────────────────────────────────────────┤                │
│  │ ob_type                     (8 bytes)   │                │
│  ├─────────────────────────────────────────┤                │
│  │ *__dict__                   (8 bytes)   │ Instance dict  │
│  ├─────────────────────────────────────────┤                │
│  │ *__weakref__                (8 bytes)   │ Weak refs      │
│  └─────────────────────────────────────────┘                │
│                                                             │
│  Minimum: 32 bytes + dict overhead                          │
│                                                             │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│ Slotted Instance (with __slots__)                           │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  class Point:                                               │
│      __slots__ = ['x', 'y']                                 │
│                                                             │
│  ┌─────────────────────────────────────────┐                │
│  │ ob_refcnt                   (8 bytes)   │                │
│  ├─────────────────────────────────────────┤                │
│  │ ob_type                     (8 bytes)   │                │
│  ├─────────────────────────────────────────┤                │
│  │ x                           (8 bytes)   │ Slot 0         │
│  ├─────────────────────────────────────────┤                │
│  │ y                           (8 bytes)   │ Slot 1         │
│  └─────────────────────────────────────────┘                │
│                                                             │
│  Total: 32 bytes (no dict overhead)                         │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## Free-Threading Object Layout (Python 3.13+)

```
┌─────────────────────────────────────────────────────────────┐
│ PyObject (Free-Threading Build)                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ┌─────────────────────────────────────────┐                │
│  │ ob_tid                      (8 bytes)   │ Owner thread   │
│  ├─────────────────────────────────────────┤                │
│  │ ob_ref_local               (4 bytes)    │ Local refcount │
│  ├─────────────────────────────────────────┤                │
│  │ ob_ref_shared              (4 bytes)    │ Shared refcount│
│  ├─────────────────────────────────────────┤                │
│  │ ob_type                     (8 bytes)   │ Type pointer   │
│  └─────────────────────────────────────────┘                │
│                                                             │
│  Biased reference counting:                                 │
│  - ob_tid: Thread that "owns" the object                    │
│  - ob_ref_local: Fast path (no atomics) for owner           │
│  - ob_ref_shared: Atomic counter for other threads          │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```
